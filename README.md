# U2AF1 Splicing Analysis (2026)

This repository contains R scripts and processed rMATS data comparing **U2AF1 Q157R vs WT** splicing events.

## Data Overview
- **rMATS Outputs:** Contains Skipped Exon (SE) analysis using both Junction Counts (JC) and Junction Counts + Exon Counts (JCEC).
  - `SE.MATS.JCEC_bao.txt` & `SE.MATS.JC_bao.txt`
  - `SE.MATS.JCEC_sg_new.txt` & `SE.MATS.JC_sg.txt`
- **Summary:** `2026_02_19_summary_sg_analysis.txt` provides a high-level overview of significant events.

## Analysis Scripts (R)
- **Event Plotting:** `2026_02_17_u2af1_splicing_event_plot_bao.R` - Main script for visualizing specific splicing changes.
- **Scatter & Logos:** `2026_02_19_sg_scattor_plot_sequence_logos.R` - Generates scatter plots for inclusion levels and sequence motif logos.
- **Summary Plots:** `2026_02_19_sg_splicing_events_summary_plot.R` - Bar charts/summaries of differential splicing types.
